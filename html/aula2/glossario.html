Glossário de Tags

<h1>

↑ Isto é um título. Ele é o maior elemento de texto que você pode ter na página sem CSS, e é amplamente usado em, como o nome do elemento já diz, títulos.

<p>

é um parágrafo ordinário. Ele é como um elemento de texto qualquer, mas ele é um elemento que ocupa uma linha própria

<br>

uma simples tag que cria uma quebra de linha.

<hr>

Cria uma quebra de linha, mas com uma linha desenhada cortando a página.

<ul>

Cria uma lista não-ordenada.

<ol>

Cria uma lista ordenada.

<a>

Cria um "Link"(cujo nome real é anchor) para outra página.

<img>

Adiciona uma imagem à página.
